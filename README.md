# Algorithm Strategies Mini Project
### ADA Lab Assignment 2 — Divide & Conquer | Greedy | Dynamic Programming | TSP

---

## Problem Context

Real-world systems constantly face optimization challenges — delivery routing, resource allocation, data sorting, and sequence matching. This project implements and benchmarks five core algorithmic paradigms to understand their practical efficiency and when to apply each.

---

## Algorithms Implemented

| Task | Algorithm | Paradigm | Complexity |
|------|-----------|----------|------------|
| 1 | Merge Sort | Divide & Conquer | O(n log n) |
| 1 | Binary Search | Divide & Conquer | O(log n) |
| 2 | Bubble / Merge / Quick Sort | Comparison | O(n²) to O(n log n) |
| 3 | Activity Selection | Greedy | O(n log n) |
| 3 | Fractional Knapsack | Greedy | O(n log n) |
| 4 | 0/1 Knapsack | Dynamic Programming | O(n × W) |
| 4 | Longest Common Subsequence | Dynamic Programming | O(m × n) |
| 5 | TSP Brute Force | Exhaustive Search | O(n!) |
